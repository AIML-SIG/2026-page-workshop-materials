## About

This repository contains all hands-on materials for the PAGE 2026 AI/ML Satellite Workshop. The workshop combines live coding sessions in R and Julia with a panel discussion on regulatory considerations for AI/ML in pharmacometrics and clinical pharmacology.

## Repository Structure

```

├── 00_data/                          # Datasets used across all hands-on sessions
```
